METODO MAIN ===>

É O PONTO DE ENTRADA PARA A EXECUÇÃO DE QUALQUER PROGRAMA JAVAO MÉTODO main É O PONTO DE ENTRADA DE TODO PROGRAMA JAVA. É A PRIMEIRA PARTE DO CÓDIGO QUE O COMPILADOR EXECUTA QUANDO O PROGRAMA É INICIADO. SEM ELE, O PROGRAMA NÃO CONSEGUE RODAR COMO UMA APLICAÇÃO INDEPENDENTE. 


PARA QUE SERVE O METODO MAIN? ---> 

1 - PONTO DE INICIALIZAÇÃO:
        O MÉTODO main DEFINE ONDE O PROGRAMA COMEÇA A EXECUTAR. TUDO QUE VOCÊ QUER QUE O PROGRAMA FAÇA DEVE SER INICIADO DENTRO DELE OU POR MÉTODOS CHAMADOS A PARTIR DELE.

2 - CONFIGURAÇÃO INICIAL:
        VOCÊ PODE USAR O main PARA INICIALIZAR OBJETOS, CONFIGURAR VARIÁVEIS OU CHAMAR OUTROS MÉTODOS PARA EXECUTAR TAREFAS ESPECÍFICAS.

3 - INTERAÇÃO COM O USUÁRIO:
        O MÉTODO main TAMBÉM É RESPONSÁVEL POR INTERAGIR COM O USUÁRIO DIRETAMENTE, POR MEIO DE ENTRADAS E SAÍDAS NO CONSOLE.


COMO FUNCIONA A ESTRUTURA DO MÉTODO main? ---> 

A DECLARAÇÃO PADRÃO DO MÉTODO main É:

public static void main(String[] args) {
    // SEU CÓDIGO VAI AQUI
}


QUEBRANDO A DECLARAÇÃO:

1 - public:
        SIGNIFICA QUE O MÉTODO PODE SER ACESSADO DE QUALQUER LUGAR, NECESSÁRIO PARA QUE A JVM (MÁQUINA VIRTUAL JAVA) POSSA ENCONTRÁ-LO E EXECUTÁ-LO.

2 - static:
        PERMITE QUE O MÉTODO SEJA CHAMADO SEM QUE UM OBJETO DA CLASSE SEJA CRIADO.
        COMO O MÉTODO main É O PRIMEIRO A SER EXECUTADO, ELE NÃO PODE DEPENDER DE UMA INSTÂNCIA.    

3 - void:
        INDICA QUE O MÉTODO NÃO RETORNA NENHUM VALOR.

4 - String[] args:
        UM ARRAY DE STRINGS QUE CONTÉM ARGUMENTOS PASSADOS NA LINHA DE COMANDO QUANDO O PROGRAMA É EXECUTADO.
