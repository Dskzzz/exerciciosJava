PACOTES ===>

SÃO UMA FORMA DE ORGANIZAR AS CLASSES E INTERFACES DENTRO DE UM PROJETO. ELES FUNCIONAM COMO PASTAS OU DIRETÓRIOS QUE AGRUPAM ARQUIVOS RELACIONADOS, TORNANDO O CÓDIGO MAIS ESTRUTURADO E FÁCIL DE GERENCIAR.

SERVEM PARA --->

ORGANIZAÇÃO DO CÓDIGO:

AJUDAM A MANTER UM PROJETO LIMPO E ORGANIZADO, ESPECIALMENTE EM APLICAÇÕES GRANDES COM MUITAS CLASSES.

EVITAR CONFLITOS DE NOMES:

DOIS DESENVOLVEDORES PODEM CRIAR CLASSES COM O MESMO NOME. OS PACOTES GARANTEM QUE ESSAS CLASSES FIQUEM ISOLADAS.

CONTROLE DE ACESSO:

COMBINADOS COM MODIFICADORES DE ACESSO (COMO public, protected, SEM MODIFICADOR OU private), OS PACOTES PODEM LIMITAR O QUE É VISÍVEL ENTRE DIFERENTES PARTES DO PROGRAMA.

REUTILIZAÇÃO DE CÓDIGO:

FACILITAM A IMPORTAÇÃO E REUTILIZAÇÃO DE CLASSES E MÉTODOS DE OUTROS PACOTES.

COMO USAR PACOTES EM JAVA --->

1 - DECLARAÇÃO DE UM PACOTE:
        A PRIMEIRA LINHA DE UM ARQUIVO JAVA DEVE INDICAR O NOME DO PACOTE: 
        
               EXEMPLO: package meuPacote;

2 - IMPORTANDO UM PACOTE:
        PARA UTILIZAR CLASSES DE OUTRO PACOTE, VOCÊ PRECISA IMPORTÁ-LO: 
        
               import meuPacote.MinhaClasse;

2 - IMPORTANDO UM PACOTE:
        OU USAR UM IMPORT GENÉRICO PARA TODAS AS CLASSES DO PACOTE: 
        
               import meuPacote.*;

3 - USANDO BIBLIOTECAS PADRÃO:

        O JAVA TEM VÁRIOS PACOTES PRÉ-DEFINIDOS, COMO:
                java.util: PARA ESTRUTURAS DE DADOS COMO LISTAS E MAPAS.
                java.io: PARA OPERAÇÕES DE ENTRADA E SAÍDA.
                java.math: PARA OPERAÇÕES MATEMÁTICAS.

CLASSES ===>

CLASSES SÃO A BASE DA PROGRAMAÇÃO ORIENTADA A OBJETOS (POO) EM JAVA. ELAS SÃO COMO MOLDES OU BLUEPRINTS QUE DEFINEM AS PROPRIEDADES (ATRIBUTOS) E COMPORTAMENTOS (MÉTODOS) DOS OBJETOS QUE SERÃO CRIADOS A PARTIR DELAS.

SERVEM PARA --->

1 - ORGANIZAR O CÓDIGO:

        CLASSES AGRUPAM DADOS E MÉTODOS RELACIONADOS EM UMA ÚNICA ESTRUTURA, FACILITANDO A LEITURA E MANUTENÇÃO DO CÓDIGO.

2 - CRIAR OBJETOS:

        EM JAVA, OBJETOS SÃO INSTÂNCIAS DE CLASSES. POR EXEMPLO, SE UMA CLASSE DEFINE UM CARRO, OS OBJETOS SÃO OS CARROS ESPECÍFICOS COM SUAS PRÓPRIAS CARACTERÍSTICAS.

3 - REUTILIZAÇÃO DE CÓDIGO:

        VOCÊ PODE USAR A MESMA CLASSE PARA CRIAR DIVERSOS OBJETOS SEM PRECISAR REESCREVER O CÓDIGO.

4 - ENCAPSULAMENTO:

        CLASSES PERMITEM OCULTAR OS DETALHES INTERNOS DOS OBJETOS, EXPONDO APENAS O QUE É NECESSÁRIO ATRAVÉS DE MÉTODOS PÚBLICOS.


COMO DECLARAR E USAR CLASSES EM JAVA? --->

1 - DECLARAÇÃO DE UMA CLASSE:   
        UMA CLASSE É DEFINIDA USANDO A PALAVRA-CHAVE class:
            EXEMPLO: 
            
                        public class MinhaClasse{
                        // ATRIBUTOS
                        int idade;
                        String nome;

                        // MÉTODOS
                        void mostrarNome() {
                            System.out.println("O NOME É: " + nome);
                        }
                    }


2 - CRIANDO UM OBJETO:   
        VOCÊ USA O OPERADOR new PARA INSTANCIAR UM OBJETO:
            EXEMPLO: 

                        MinhaClasse obj = new MinhaClasse();
                        obj.nome = "Douglas";
                        obj.mostrarNome();

METODO MAIN ===>

É O PONTO DE ENTRADA PARA A EXECUÇÃO DE QUALQUER PROGRAMA JAVA 

A ASSINATURA: public static void main(String[] args) É OBRIGATORIA

public: TORNA O METODO ACESSIVEL EM QUALQUER LUGAR 
static: PERMITE QUE O METODO SEJA EXECUTADO SEM CRIAR UM OBJETO DA CLASSE
void: INDICA QUE O METODO NÃO RETORNA NENHUM VALOR
String[] args: REPRESENTA ARGUMENTOS PASSADOS PARA O PROGRAMA DE LINHA DE COMANDO
